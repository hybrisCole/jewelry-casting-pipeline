# US Ring Size Chart

Finger size is defined by **inner diameter** and **inner circumference**.

Half sizes step ≈ **0.4 mm** inner diameter (~1.25 mm circumference).

| US Size | Inner Diameter (mm) | Inner Circumference (mm) |
|---------|---------------------|--------------------------|
| 5 | 15.7 | 49.3 |
| 5.5 | 16.1 | 50.6 |
| 6 | 16.5 | 51.9 |
| 6.5 | 16.9 | 53.1 |
| 7 | 17.3 | 54.4 |
| 7.5 | 17.7 | 55.7 |
| 8 | 18.1–18.2 | 57.0–57.2 |
| 8.5 | 18.5 | 58.3 |
| 9 | 18.9 | 59.5 |
| 9.5 | 19.4 | 60.8 |
| 10 | 19.8 | 62.3 |

**Fixture target:** US 8 → use **18.14 mm** inner diameter as working target.

Investment casting tolerance is typically ±0.1–0.2 mm; expect mandrel sizing after cast for exact fit.
