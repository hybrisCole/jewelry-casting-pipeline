# Shrinkage Reference Table

Lost-wax casting shrinkage is **compound**: resin + investment + metal solidification (multiplicative, not additive).

**Before applying scale in Blender:** confirm metal alloy, resin brand, and whether your casting house applies its own compensation.

| Metal | Resin (example) | Combined scale factor | Notes |
|-------|-----------------|----------------------|-------|
| Sterling silver | Siraya Cast | 1.025 (~2.5%) | Verify with calipers after first cast |
| 14k yellow gold | Siraya Cast | 1.020 (~2.0%) | Ask caster if they pre-scale |
| 18k yellow gold | Siraya Cast | 1.022 (~2.2%) | Estimate — measure and update |
| Unknown / first run | Any | 1.025 | Conservative default |
